# PARA RENOMEAR MANTENHA A MONTAGEM DO INICIAL

## Comandos de configuração

### Ver o nome que está cadastrado

```AT+NAME```


### Ver a senha que está cadastrado

```AT+PSWD```

### Alterar o nome

```AT+NAME= TomateBluetooth```

### Alterar a senha

```AT+PSWD="1234"```


### Checar a data da versão do hc-05

```AT+VERSION```
